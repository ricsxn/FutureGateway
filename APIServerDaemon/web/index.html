APIServerDaemon

The APIServerDaemon executes commands prepared by the FutureGateway APIServer front-end. The front-end is responsible to get user requests from its REST interface and prepares the task execution for the APIServer which physically executes the task. The APIServer daemon retrieves the list of tasks to execute from a database table. Each task may involve a different executor identified by the task queue field 'Target'.

Actions

Below a list of actions that could be performed by the APIServerDaemon web interface.

  • View current configuration
  • Show server statistics
